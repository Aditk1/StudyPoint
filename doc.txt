1. Executive Summary
StudyPoint is a multi-tenant AI-powered SaaS platform engineered for India's autonomous higher education ecosystem. It transforms uploaded academic content into structured knowledge layers that power adaptive learning tools. This document defines a production-grade architecture, technical specifications, scalability plans, SLA matrices, database schema, API contracts, and deployment strategy.

Target Segments:

• Colleges & Universities (Tiered SaaS Pricing)

7. AI Processing Pipeline
Step 1: File Upload & S3 Storage
Step 2: Text Extraction (PyMuPDF + PaddleOCR)
Step 3: Chunking (500 tokens, 50 overlap)
Step 4: Embedding (all-MiniLM-L6-v2)
Step 5: Clustering (HDBSCAN)
Step 6: Labeling via LLM with confidence scoring
Step 7: Knowledge Graph Construction (Neo4j)
